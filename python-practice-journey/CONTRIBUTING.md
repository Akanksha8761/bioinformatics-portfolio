## How to Contribute

While this is primarily a personal learning repository, I welcome:

### 1. Suggestions for Challenges
- Open an issue with the tag `challenge-suggestion`
- Describe the concept and difficulty level
- Provide a sample problem if possible

### 2. Alternative Solutions
- Fork the repository
- Add your solution in a separate file (e.g., `solution_alternative.py`)
- Create a pull request with a clear description

### 3. Bug Fixes
- If you find any errors in the code or documentation
- Open an issue or submit a pull request with the fix

### 4. Resource Recommendations
- Share helpful learning resources
- Add them via an issue with the tag `resource`

## Guidelines

- Keep solutions beginner-friendly when possible
- Include comments explaining complex logic
- Follow PEP 8 style guidelines for Python code
- Be respectful and constructive in all interactions

## Code Style

Please follow these conventions:
- Use 4 spaces for indentation
- Follow PEP 8 naming conventions
- Add docstrings to functions
- Include comments for complex logic

## Questions?

Feel free to open an issue with the `question` tag.
